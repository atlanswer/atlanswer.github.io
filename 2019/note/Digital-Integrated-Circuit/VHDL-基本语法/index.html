VHDL 基本语法

这里只列出可能要考的语句。

并行语句

并行信号赋值 (<=)

使用运算符 <= 为端口或信号赋值。

注意信号赋值语句在顺序语句中也存在，在进程中的信号赋值语句是顺序语句，在进程外的信号赋值语句是并行语句。

条件赋值 (when...else)

优先顺序的条件判断。从第一条条件表达式开始顺序判断，直到最后一个 else。

例：实现三态门

例：实现 2-4 译码器

例：实现带使能的 2-4 译码器

选择赋值 (with...select)

同时进行条件判断。在结尾使用 when others 以包含全部情况。

例：实现七段显示译码器

进程语句 (process)

进程语句本身是并行的，其内部的语句按顺序执行，因此可以按软件编程的思想实现算法。必须为进程提供完整的敏感信号表，在敏感信号表中出现的变量和信号发生变化时进程即被激活。

例：使用进程实现优先编码器

元件例化语句 (component)

元件例化体现了 VHDL 模块化设计的思想，通过使用已经定义的元件的实例可以简化设计流程。

例：并联两个 8 bits 加法器实现 16 bits 加法器

顺序语句

顺序语句必须放在进程或子程序中，定义进程和子程序的算法，即并行语句包含顺序语句。

顺序赋值语句 (<= 和 :=)

使用运算符 <= 为端口或信号赋值，使用 := 为变量赋值。

变量的声明的赋值只能在顺序语句中进行，例如进程、函数和过程。

if-else 语句

if-else 是最常用的顺序语句，用于流程控制。

case-when 语句

case-when 语句也是顺序语句，只能在进程中使用，用来选择有明确描述的信号。

例：使用 case-when 实现多路复用器。
