学术规范与论文写作

研究活动

前沿研究性活动

  1. 解决基础科学中的理论猜想和实验问题
  2. 解决应用科学和技术难题
  3. 在新的环境、材料体系中寻找应用
  4. 生物医学领域

如何开展科研活动

  1. 夯实课程中的理论和实验基础
  2. 参与导师的创新研究项目
  3. 与导师进行学术问题讨论
  4. 广泛参与你感兴趣的研究课题的讨论（学术会议）
  5. 文献检索、阅读前沿研究内容

科技写作的内容

写作范畴

以标准格式，在科技期刊上发表的原创成果。

标准格式

  1. 作者信息
  2. 摘要（Abstract）
  3. 图（Figures）
  4. 表（Extended Data Table）

科技写作的关键

  1. 了解读者：理解接受、习语
  2. 表达清晰
  3. 科技写作中的结构和语言

写作流程

  1. 调整心态
  2. 认真准备
    1. 模仿行业文献：哪几部分、顺序、各部分篇幅、图表类型？
    2. 投稿须知
    3. 命名法（nomenclature），参考文献格式等
    4. 随时记录论文想法
    5. 团队共享
    6. 写作提纲（outlines）
    7. 文件查找、文献管理
  3. 抓紧去写
    1. 标记预留时间
    2. 设置截止日期
    3. 从最容易的部分开始 Methods，Acknowledgment，Figures，Title，Abstract
    4. 遇到问题，先做标记
    5. 选择一个解决方案，无需最优，先完成草稿
  4. 修改完善
    1. 修改时思考的问题 全部必要信息、是否需要删除、推理是否正确、内容一致、结构合理、措辞清晰、表达简单扼要、语法拼写正确、图标设计是否合理、满足投稿须知要求？
    2. 读者反馈 业内专家、领域内人士、聪敏的普通读者
    3. 反馈修改 误解问题、多名读者不懂
    4. 没有完美的论文

论文基本概念

科技论文的定义

科技论文是一种公之于众的，描述原创科研成果的书面报道。撰写必须遵循特定规范，这取决于传统、编辑惯例、科学道德以及印刷出版规程。

文献检索

引言

文献调研：

  • 读人：追踪领域内权威专家
  • 读典：精度经典文章、高被引文章
  • 读新：泛读 + 精读最新科研文章

文献检索思路

  1. 关键词搜索 Google，Web of Science，Wikipedia，知乎，SciHub 等
  2. 深入的文献调研
    1. 读人
      Google Scholar，Research Gate，Web of Science，个人主页，Quora，小木虫，Scopus
      1. 研究生时期的研究工作
      2. Researcher 后的研究方向
      3. 重要文章
      4. 不同时间段的研究方向转变
      5. 重大科学发现前后的工作内容
    2. 读典
      1. 大家都在讨论的、推荐的：实验室、同行口口相传
      2. Google Scholar 会列出相关文章、引用率排序
      3. 论文中在引言部分经常被引用的论文
      4. 其他工具
        1. Web of Science（ESI 高被引）
        2. PubMed
    3. 读新
      arXiv
      文献订阅工具：
      1. Google Scholar
      2. Publication Alert
      3. Researcher
      4. RSS 阅读器
        1. Feedly
        2. Inoreader

课程选题

本学期任务起点：

  1. 找到一个感兴趣的题目
  2. 本周开始着手去研究它
  3. 今后的课程作业都会围绕这个题目
  4. 开始可以多考虑几个方向
  5. 一般都从读文献开始

如何确定一个题目：

  1. 读好文章
  2. 跟导师聊
  3. 初步跟进：看 Wikipedia，读 review 文章，与师兄师姐讨论
  4. 前沿阶段：读 paper，关注公众号
  5. 听学术报告

选题要求：

  1. 以上提到的方法都适用
  2. 希望通过文献阅读、提升自己某个方面新的认识
  3. 终稿建议仿写英文论文
  4. 每周大家根据课堂内容，修改自己的论文，并以此参与课堂讨论
  5. 不能拷贝别人的东西
  6. 不要求理论、实验创新，但是认识可以创新，提出评价，或者某方面自己的深刻认识
  7. 最终稿没有字数限制
  8. 建议由浅入深，经过本学期的课程学习，能把问题讲清楚

斟酌标题

撰写摘要

摘要定义

微型版论文（miniature version）—— structured abstract

  1. 研究目标和范围 Background
  2. 描述研究方法 Method
  3. 总结研究结果 Result
  4. 给出结论 Conclusion

摘要类型

  • 信息型摘要（Informative abstract）
  • 指示型摘要（Indicative/descriptive abstract）

作业：撰写摘要

  1. 研究背景和问题：通过阅读文献等方法，确立自己的研究问题
  2. 根据我们讨论的摘要的写法，形式性地撰写一则摘要，内容尽可能贴近现实，少量虚构也可以
  3. 注意三点：问题描述、研究方法、研究结果的表达
