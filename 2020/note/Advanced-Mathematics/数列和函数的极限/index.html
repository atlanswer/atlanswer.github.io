数列和函数的极限

数列

Wikipedia

数列指的是一列数。“一列”指出其至少具有一维，即有序性。我们用正整数为每一项编号，\(a_1,a_2,\dotsc,a_n,\dotsc\) 表示无穷数列，简称数列，记作 \({a_n}\)。

如果有通项 \(a_n\) 的解析式，那么数列可以看成定义在正整数集上的函数 \(f:\Z^+\to y\)：\[f(n)=a_n,\ n=1,2, \dotsc\]

数列的极限

Wikipedia

\(\epsilon-N\) 定义

\[ \def\arraystretch{1.5} \begin{array}{c} \lim\limits_{n\to\infty}a_n=l \iff \\ \forall\epsilon>0,\ \exist N>0,\ \forall n>N\implies |a_n-l|<\epsilon \end{array} \]

若极限 \(l\) 存在，则称数列收敛 (convergent)，否则发散 (divergent)。

数列极限的性质

  1. 夹逼定理 (squeeze theorem)

    \[ \def\arraystretch{1.5} \begin{array}{l} \exist N_0\in\Z^+,\text{ if } a_n\leqslant b_n\leqslant c_n,\forall n>N_0,\\ \text{ and}\lim\limits_{n\to\infty} a_n=\lim\limits_{n\to\infty}b_n=l,\\ \text{ then}\lim\limits_{n\to\infty}c_n=l \end{array} \]

  2. 不等式：极限 \(\implies\) 数列

  3. 2 的推论：由极限判断数列符号

  4. 不等式：数列 \(\implies\) 极限

  5. 四则运算
    注意四则运算的前提是两个数列都收敛。
    四则运算可以推广到有限个数列的四则运算，但对无限次运算不一定成立。

  6. 数列收敛的充要条件是所有子数列都收敛，所有极限与原数列极限相同。

证明数列发散的方法

  1. 利用 6，由子数列发散或收敛与不同值证明原数列发散。

证明数列收敛的方法

  1. 利用定义。\(N\) 的取值决定于 \(\epsilon\)，不需要求得最小的 \(N\)。
  2. 利用实数域的完备性：单调有界数列必有极限，但没有给出求极限的方法。
  3. 夹逼定理。
  4. 通过四则运算得到重要极限的组合。

重要极限

例：证明 \(\lim\limits_{n\to\infty}\left(1+\frac{1}{n}\right)^n\) 存在。

证明：

证明 \(a_n=\left(1+\frac{1}{n}\right)^n\) 递增，即 \[ \begin{aligned} \left(1+\frac{1}{n}\right)^n&<\left(1+\frac{1}{n+1}\right)^{n+1}\\ \implies\sqrt[n+1]{\left(1+\frac{1}{n}\right)^n}&<\frac{n+2}{n+1} \end{aligned} \] 在不等关系中遇到 \(\sqrt[n]{x^n}\)，即几何平均值，可以考虑使用均值不等式： \[ \sqrt[n+1]{\left(1+\frac{1}{n}\right)^n} <\frac{n\left(1+\frac{1}{n}\right)+1}{n+1} =\frac{n+2}{n+1} \] which is amazing, 为什么不等式在这里恰好合适呢，我暂时没有头绪。

根据实数域的完备性，还需要给出数列有界的条件，试证明 \(a_n<4\)： \[ 1+\frac{1}{n}<\sqrt[n]{4} \] 在这里直接使用均值不等式方向不对，所以需要交换一下位置。 \[ \begin{gathered} \frac{1}{4}<\left(\frac{n}{n+1}\right)^n\\ \implies\sqrt[n]{\frac{1}{4}}= \sqrt[n]{\frac{1}{2}\sdot\frac{1}{2}\cdot 1\dotsm 1} <\frac{\frac{1}{2}+\frac{1}{2}+n-2}{n} =\frac{n-1}{n}\\ =1-\frac{1}{n}<1-\frac{1}{n+1}=\frac{n}{n+1} \end{gathered} \] 这里需要拼一下根号下的乘积，应该只有这种组合可以简单地得到结论，属于技巧性操作。

由例题我们得知 \(\lim\limits_{n\to\infty}\left(1+\frac{1}{n}\right)^n\) 存在，这个极限值记为 \(e\)： \[\lim\limits_{n\to\infty}\left(1+\frac{1}{n}\right)^n=e\] 也就是著名的自然对数的底数，这一极限是 \(e\) 的定义式。

\(e\approx2.718281828\)，小数点后九位非常有规律所以其近似值并不难记。相信大家看过一些关于 \(e\) 意义的精彩解读，它无疑是数学中最重要的常数之一，尤其对理工科而言，它是联系不同域的桥梁。

例：求 \(\lim\limits_{n\to\infty}\left(1+\frac{2}{n+1}\right)^n\)

解：观察通项形式，思路应该是通过四则运算将其分解为重要极限。 \[ \begin{aligned} \lim\limits_{n\to\infty}\left(1+\frac{2}{n+1}\right)^n &=\lim\limits_{n\to\infty}\left(1+\frac{1}{\frac{n+1}{2}}\right) ^{\frac{n+1}{2}\cdot 2-1}\\ &=\lim\limits_{n\to\infty}\left[\left(1+\frac{1}{\frac{n+1}{2}}\right) ^\frac{n+1}{2}\right]^2\left(1+\frac{2}{n+1}\right)^{-1} \end{aligned} \] 注意每一项的极限都存在，因此 \[ \lim\limits_{n\to\infty}\left(1+\frac{2}{n+1}\right)^n=e^2 \]

函数的极限

  • 趋于无穷：\(\epsilon-X\) 定义

    \[ \def\arraystretch{1.5} \begin{array}{c} \lim\limits_{x\to+\infty}f(x)=A \iff \\ \forall\epsilon>0,\ \exist X>0,\ \forall x>X\implies |f(x)-A|<\epsilon \end{array} \]

    \[ \def\arraystretch{1.5} \begin{array}{c} \lim\limits_{x\to-\infty}f(x)=A \iff \\ \forall\epsilon>0,\ \exist X>0,\ \forall x<-X\implies |f(x)-A|<\epsilon \end{array} \]

  • 趋于有限值：\(\epsilon-\delta\) 定义

    \[ \def\arraystretch{1.5} \begin{array}{c} \lim\limits_{x\to x_0}f(x)=A \iff \\ \forall\epsilon>0,\ \exist \delta>0, \ |f(x)-A|<\epsilon\text{ if } 0<|x-x_0|<\delta \end{array} \]

    注意：函数极限存在与否与函数在 \(x_0\) 点是否有定义没有关系，\(x\) 落在 \(x_0\) 的去心邻域中。

  • 单侧极限：

    • 左极限：

      \[ \def\arraystretch{1.5} \begin{array}{c} \lim\limits_{x\to x_0^-}f(x)=A \iff \\ \forall\epsilon>0,\ \exist \delta>0, \ |f(x)-A|<\epsilon\text{ if } x_0-\delta<x<x_0 \end{array} \]

    • 右极限：

      \[ \def\arraystretch{1.5} \begin{array}{c} \lim\limits_{x\to x_0^+}f(x)=A \iff \\ \forall\epsilon>0,\ \exist \delta>0, \ |f(x)-A|<\epsilon\text{ if } x_0<x<x_0+\delta \end{array} \]

函数极限的性质

  1. 极限存在定理

    \[ \def\arraystretch{1.5} \begin{array}{l} \lim\limits_{x\to x_0}f(x)=A \iff\\ \lim\limits_{x\to x_0^-}f(x) =\lim\limits_{x\to x_0^+}f(x) =A \end{array} \]

    左右极限存在且相等 \(\iff\) 函数极限存在
